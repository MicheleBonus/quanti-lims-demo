Was sich ändert

Vorher (fehlerhaft)

  • Validierung prüft nur mges ≥ target
  • TA kann 2,2 g + 7,8 g wiegen → Gwahr = 22 %
  • Kein Live-Feedback während Eingabe
Nachher (korrekt)
  • Zeigt „Max. mges" dynamisch basierend auf ms
  • Validierung prüft mges ≤ ms × peff / pmin
  • Gwahr live berechnet + rot/grün markiert
  • Orientierungswert mges-Richtwert bleibt als Hinweis
